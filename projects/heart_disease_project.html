Heart Disease Prediction

Heart disease is a leading cause of death worldwide, making early detection and accurate prediction crucial. This project analyzes the performance of four machine learning algorithms—Decision Trees, K-Nearest Neighbors, Artificial Neural Networks, and Hopfield Networks—on a heart disease dataset. By comparing accuracy, efficiency, and interpretability, the project highlights the most effective approaches for supporting early diagnosis and personalized treatment in healthcare.

Technologies Used

  • Python
  • Scikit-learn
  • Pandas
  • NumPy
  • Matplotlib
  • Data Visualization
  • Machine Learning

Artificial Nerual Networks

98% Overall Accuracy

Key Features:

  • Used Multilayer Perceptron (MLP) classifier from scikit-learn.
  • MLP structure: input layer (features), two hidden layers (30 neurons each), output layer (predictions).
  • Key mechanisms:
    • Forward propagation — data moves through layers with weighted sums and activation functions (ReLU).
    • Loss calculation and backpropagation — compute gradients to update weights and biases.
    • Optimization — Adam optimizer with learning rate 0.001.

K-Nearest Neighbors

83.4% Overall Accuracy

Implementation Details:

  • Implemented K-Nearest Neighbors (KNN) for heart disease prediction.
  • Used Matplotlib and Seaborn to create a correlation heatmap for visualizing relationships between features and the target.
  • Identified influential attributes:
    • Positive correlation — chest pain (cp), maximum heart rate (thalach).
    • Negative correlation — ST depression (oldpeak), exercise-induced angina (exang).

Decision Trees

87% Overall Accuracy

Technical Approach:

  • Implemented a custom Decision Tree model using pandas, NumPy, and matplotlib only.
  • Selected features at each node by minimizing weighted Gini Impurity.
  • Recursively split nodes until purity or a maximum depth of 4 (to prevent overfitting).
  • Visualized the resulting tree using matplotlib.
