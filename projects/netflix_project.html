Netflix Title Analysis

The goal of this project was to analyze the Netflix dataset from Kaggle to uncover insights about the content available on the platform. This involved data cleaning, exploration, and visualization.

I explored questions such as:

  • What is the distribution between Movies vs TV Shows?
  • What were the number of titles added per year?
  • What were the top countries producing titles?
  • What were the most common genres?

Technologies Used

  • Python
  • Pandas
  • NumPy
  • Matplotlib
  • Seaborn
